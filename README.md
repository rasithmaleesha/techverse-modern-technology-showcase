# 💻 TechVerse – Modern Technology Showcase Website

> **Discover the Future of Technology. 🚀**

A modern and responsive **technology showcase website** inspired by Apple's clean design philosophy. TechVerse allows users to explore premium technology products from leading global brands through an elegant, interactive, and visually engaging experience.

---

## ✨ Features

- 🎨 Apple-Inspired Modern UI
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🔍 Product Search
- 📂 Category Browsing
- 💎 Glassmorphism UI
- ⚡ Smooth Animations
- 🖼️ Featured Product Showcase
- 📊 Product Comparison Section
- 📧 Newsletter Subscription
- 📞 Contact Page
- ℹ️ About Page
- 🚀 Fast & Clean User Experience
